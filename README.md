# 迁镜（ShotSwift）

产品名称：迁镜（ShotSwift）

项目全称：爆款结构迁移引擎：样例驱动的视频结构迁移与 AI 改片 Agent

团队名称：随机队友生成队

## 项目简介

本项目面向短视频创作场景，目标是构建一个“样例驱动的视频结构迁移工作台”。

用户可以上传一条爆款样例视频，系统自动拆解其中的脚本结构、节奏结构和包装结构，再结合用户输入的新主题、新卖点和素材，生成新的短视频脚本、分镜、时间线草案、包装建议，并识别素材缺口和给出补全方案。

第一阶段聚焦种草 / 拔草类短视频（当前 Demo 垂类：新手养猫怎么选猫粮 / pet_food），优先跑通从样例解析、结构拆解、结构迁移、素材缺口识别到结果生成的 P0 闭环。
